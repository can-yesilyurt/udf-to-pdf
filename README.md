# UDF-to-PDF

**UYAP .udf dosyalarını tarayıcıda görüntüleyin ve PDF'e dönüştürün.**

View and convert Turkish UYAP court documents (.udf) to PDF — entirely in your browser. No server, no upload, no data leaves your device.

## Features

- 📄 Drag-and-drop UDF file viewer
- 🔒 100% client-side — files never leave your device
- 📥 One-click PDF export
- 🖨️ Print support
- ✅ Digital signature detection

## Usage

Just open `index.html` in any modern browser, or visit the hosted version.

## Hosting

This is a single static HTML file. Deploy anywhere:

- **GitHub Pages** — push and enable Pages in repo settings
- **Netlify / Cloudflare Pages** — drag and drop
- **Any web server** — just serve the file
